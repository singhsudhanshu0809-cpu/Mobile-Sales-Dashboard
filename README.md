# Mobile-Sales-Dashboard

## 📌 Project Overview

This project showcases an **interactive Mobile Sales Dashboard** built using **Microsoft Power BI**. The dashboard provides insights into mobile sales performance, transaction trends, customer ratings, and payment methods across different cities and months.

This project was created as part of my learning journey in **Power BI**, guided by **Satish Dhawale Sir's Skill Course**, where I learned how to design professional and interactive dashboards.

## 🚀 Key Features

* Interactive **sales analysis dashboard**
* **Dynamic filters (slicers)** for:

  * Mobile Model
  * Payment Method
  * Brand
  * Month selection
* **Geographical visualization** of sales by city
* Monthly **total quantity trend analysis**
* **Transaction distribution by payment method**
* **Customer rating analysis**
* **Brand-wise sales and transaction comparison**
* **Mobile model performance analysis**
* **Day-wise sales performance insights**

## 📈 Dashboard Insights

The dashboard helps analyze:

* Total Sales Revenue
* Average Mobile Price
* Total Quantity Sold
* Number of Transactions
* City-wise sales distribution
* Payment method usage (UPI, Debit Card, Credit Card)
* Brand and model performance
* Sales patterns across days and months
